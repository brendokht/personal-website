## Summary

This group project was a capstone project for my managing software projects class. We created
a vision document, created user stories, and created a cost estimate for this project, as per the
requirements. We decided to challenge ourselves by creating a _Discord_ like project-management
application using Electron and an ASP.NET Core Web API. Within this application, you can register,
add friends, create servers, just like _Discord_. The main difference is that you can add projects,
sprints and tasks per server created, so you can manage the various projects you might have.
